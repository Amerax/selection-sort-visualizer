# selection-sort-visualizer

An interactive GUI application built in Python that visualizes the step-by-step mechanics of the selection sort algorithm.

I made this because I usually have a hard time visualizing or thinking about certain sorting algorithims and this basically allows you to see it and helps understand it better. 
To use this code you can run the demo and put in comma seperated numbers and then press enter. Then you can look at the steps and see how the computer and code runs. 
This is very intuitive and nice for those who struggle with sorting algorithims. 

The code works by perfomring the same selection sort algoritim and updating the values and then therefore the gui elements like the bar graph colors and what not. 
The gui code is written at the bottom. 

You can simply run the code here without any download or libary installs needed!
